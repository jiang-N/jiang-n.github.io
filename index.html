使用Hugo搭建博客的记录

Hugo简介 Hugo是使用Go语言实现的一个博客生成器 是世界上最快的博客生成器 安装Hugo 下载页面 去下载页面下载hugo_xxx_Windows-64bit.zip 解压，把hugo.exe存放到创建好的hugo目录下 把上一步的路径添加到PATH中 在终端输入hugo version如果看到一个版本号，说明安装成功 hugo version 搭建博客 在终端输入hugo new site quickstart创建一个新网站，quickstart可以替换成自己github的仓库名 hugo new site quickstart 执行以下命令添加一个主题 cd quickstart git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke 执行完上一步后执行命令 echo 'theme = "ananke"' >> config.toml 执行命令，创建你的第一篇博客 hugo new posts/my-first-post.md 执行以下命令来开启Hugo服务器 hugo server -D 配置博客 打开config.
